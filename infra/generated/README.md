# infra/generated/ — DO NOT HAND-EDIT

Terraform in this folder is **generated from `platform.json`** by
`ai/iac_generator.py` and committed so every change is a reviewable diff.

To change infrastructure, edit `platform.json` and regenerate — never edit these
files directly, or your change will be overwritten on the next generation.

Each provider's stack must pass the validation gauntlet (fmt → validate → plan →
policy → contract → destroy-guard) before it is applied. See Part C of the docs.
